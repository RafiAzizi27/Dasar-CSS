Font Style

css dapat mengatur style dari teks yang ada pada halaman web yang dibuat. Terdapat beberapa property yang dapat digunakan untuk mengatur style teks yaitu sebagai berikut.

  • text-align adalah salah satu property pada css, sedangkan fungsinya untuk membuat perataan pada text
  • color adalah digunakan untuk memberikan warna pada teks
  • letter-spacing adalah untuk mengatur jarak antar karakter
  • word-spacing digunakan untuk antar kata
